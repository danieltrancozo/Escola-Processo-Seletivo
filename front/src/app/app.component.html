Escola Processo seletivo

Apresentação

Nesse projeto existem algumas falhas de apresentação visual devido ao fato de esta ser minha primeira tentativa de criar uma aplicação com angular.

Observações

  • Devido as especificaçõesdo documento do processo seletivo pedindo que a api no backend fosse um projeto separado desse projeto no front, eu recebi muitos erros CORS enquanto testava a integração no modo de navegação comum, mesmo quando o Postman inicava que a api estava funcionando normalmente. Caso alguma tabela, não esteja carregando, siga as instuções do arquivo readme.md na pasta escola e inicialize o navegador chrome no modo de segurança web desabilitada.

  • O campo de texto usado para cadastro também é necessário para as operações de atualização de elementos nas tabelas.

  • O framework no qual a aplicação foi feita está na versão 14 pois eu não consegui encontrar um modo de instalar a versão 7 requerida a tempo de refazer a aplicação para a entrega.

Agradecimento

Agradeço pela oportunidade de realizar esse processo seletivo e pela extenção do prazo de entrega que me foi dada. Espero que meu projeto esteja de acordo com as suas expectativas e que eu possa estar trabalhando com vocês em breve.
